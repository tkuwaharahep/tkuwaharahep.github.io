自分が忘れたとき用の設定を諸々と載せておく。

Visual Studio Code + LaTeX

ExtensionにLaTeX Workshopのみを導入している。 vscode左下の設定マークから、Setting（並びに右上マーク）からsetting.jsonの編集を行う（setting.jsonの設定は参照している）。 ユーザースニペットは、latex.jsonで編集可である。

Visual Studio Code + GitHub

いつの間にか1に、VS CodeだけでGitHubとの連携ができるようになっていた。 Extensionは、Git History (by Don Jayamanne) とGitHub Pull Requests and Issues (by GitHub) ぐらい。 VS CodeとGitHubの連携Tipsに関してはTipsにまとめられているものを参照するもよし。

Latexdiff + Compile Error

latexdiffは数式にalign環境を用いるとエラーを吐く。 latexdiffのコマンドはシェルになっているので、直接編集することも可能である。

my $ARRENV='(?:aligned|align|array|[pbvBV]?matrix|smallmatrix|cases|split)';


または、オプションとして次を使う。

latexdiff --math-markup=1 old.tex new.tex > diff.tex


--math-markupは、デフォルトでは2で数が小さいほど、latexdiffでの数式の処理が雑になる。alignを使ったtexでのエラーもなくなる。
